 Headphones

A clean, modern, responsive landing page design for a fictional Headphones brand. The project demonstrates HTML5, CSS3, responsive design techniques, and adaptive styling for different screen sizes.

Features
Fully Responsive Layout

The page automatically adjusts to different screen sizes:

Desktop version (default)

Tablet layout

Mobile version (activated at screen width 480px or less)

Smooth Interactions

Links hover/active color: #FF6565

Button hover/active:

opacity: 0.9;

 Clean Layout With Max-Width Constraint

All content is contained inside a central wrapper:

max-width: 1000px;
margin: 0 auto;


This ensures perfect alignment on all devices.

Modern UI

Hero banner with background image overlay

Header navigation

Call-to-action button

Simple and elegant typography

Focus on accessibility and readability

Responsive Behavior
Mobile Version (≤ 480px)

When the screen width reaches 480px or smaller:

Navigation links stack or collapse depending on implementation

Heading font sizes scale down

Text becomes center-aligned

Margins and paddings adjust for smaller screens

CTA button becomes full width

Image fully adapts to viewport
