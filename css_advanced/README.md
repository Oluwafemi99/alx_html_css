Personal Profile Webpage

A simple, responsive personal profile webpage built with HTML5 and CSS3.
This project demonstrates clean semantic HTML structure, basic styling, tables, navigation, links, and responsive design suitable for mobile devices.

Project Overview

This webpage introduces the author and showcases basic HTML elements such as:

Semantic tags (header, main, article, aside, footer)

Headings (h1, h2, h3)

Paragraphs

Hyperlinks

A data table

Navigation menu

Responsive design using CSS

External stylesheet linking (base.css, styles.css)

The design is optimized for smartphones using the class:
works_on_smartphone

Files Included
1. index.html

The main webpage containing:

Header with navigation

Article section with personal information

Table displaying sample data

Aside placeholder for future comments

Footer with author credit

2. base.css

This file contains foundational and global styling, such as:

Reset styles

Typography rules

Global layout helpers

3. styles.css

This file includes custom styling for:

Page layout

Navigation menu

Table formatting

Article and aside structure

Footer presentation

HTML Structure Breakdown
<header>

    <article>

        Table with sample user data

    <aside>
        Placeholder text for future comment section
    </aside>
</main>

<footer>
    Author credit and external link
</footer>

 How to Run the Project

Clone or download the project folder

Ensure all files (index.html, base.css, styles.css) are in the same directory

Open index.html in your browser

Done! 

 Customization

You can personalize the following parts:

Replace <YOUR NAME> with your actual name in the footer

Update the footer link (ANY LINK)

Modify colors and layout in styles.css

Add real comment functionality inside the <aside> element
